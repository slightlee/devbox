# DevBox

开发者日常高频工具集合，即开即用，无需安装。

## 包含工具

| 工具 | 说明 |
|------|------|
| Base64 编解码 | 文本与 Base64 互相转换，支持 UTF-8 |
| 图片 Base64 | 图片与 Base64 互转 |
| JSON 工具 | 格式化、压缩、语法校验 |
| 时间戳转换 | Unix 时间戳与日期时间互转 |
| JWT 解析 | 解码 JWT Token 的 Header/Payload/Signature |
| 哈希计算 | MD5 / SHA-1 / SHA-256 / SHA-512 |
| URL 编解码 | URL 编码与解码 |
| UUID 生成 | 批量生成 UUID v4 |
| 颜色转换 | HEX / RGB / HSL 互转 |
| 正则测试 | 实时正则表达式匹配测试 |
| 字符统计 | 字符数、词数、行数、字节数统计 |

## 特性

- 纯前端，无需后端服务
- 无框架依赖，原生 HTML / CSS / JS
- 响应式设计，支持移动端
- 无障碍友好（ARIA、键盘导航、跳过链接）
- 暗色模式支持

## 使用

直接打开 `index.html` 即可使用，或部署到任意静态托管服务。

```bash
# 本地预览
npx serve .
# 或
python3 -m http.server 8080
```
